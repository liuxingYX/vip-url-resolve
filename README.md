# VIP Video Resolve

Resolve vip videos url to a m3u8 link and play it


## Quick Start

### Prerequisites

```
	python3
	pip3 install flask
	express
```

### Example

```
	cd backend
	python3 restAPI.py
```

```
	cd frontend
	npm install -d express
	node app.js
```
then open [http://localhost:5000/](http://localhost:5000/) you will find the blew example
