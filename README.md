# VIP Video Resolve

Resolve vip videos url to a m3u8 link and play it


## Quick Start

### Prerequisites

```
	python3
	pip3 install flask
```

### Example

```
	cd backend
	python3 restAPI.py
```

then open [http://localhost:6789/](http://localhost:6789/) you will find the blew example
